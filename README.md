# Tabular data — animated employee records

A single-file HTML table (40 rows × 8 columns) styled with the **GeoServe** maritime design system.

## Features

- **Staggered load animation** — rows slide and fade in one after another on "Load data".
- **11 selectable row hover effects** via the "Hover effects" menu:
  Basic hover strip, Center fill, Cursor spotlight, Directional sweep, Accent bar + indent,
  Lift (font grows, no layout shift on neighbours), Subtle zoom, Underline reveal,
  Shimmer sweep, Split border, and Row glow.
- **GeoServe theme** — Inter typeface, brand-blue (`#1852FE`) primary action, card surfaces,
  status chips (Active / Remote / On leave / Contract), and the GeoServe logo in the top bar.

## Run locally

Open `index.html` directly, or serve the folder:

```bash
npx serve -p 7885 .
```

Then visit http://localhost:7885
